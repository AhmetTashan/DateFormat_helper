# Date Format Helper
`New member of the old world`

Usage
-----

Simple use:
```php
 $date = date('Y-m-d H:i:s');
 $qdate($date, 'normal');
```

Format
-----
 * normal = 18 Ağustos 2016 / varsayilan
 * day = 18 Ağustos, Perşembe
 * all = 18 Ağustos 2016, Perşembe
 * clock = saat:dakika
 * rss = Thu, 18 Aug 2016 00:00:00 +0200
